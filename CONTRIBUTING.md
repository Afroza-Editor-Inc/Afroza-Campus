# Contributing — Afroza Campus

Merci de contribuer !

Processus:
1. Fork & branch (naming: feat/xxx, fix/xxx, chore/xxx)
2. Ouvrir une Pull Request descriptive
3. Respecter les conventions de commit et linting

Règles rapides:
- Tests unitaires pour toute logique métier
- Documenter toute API nouvelle dans `docs/api.md`
- Respecter la stack validée (ne pas ajouter de nouvelles technologies sans approbation)

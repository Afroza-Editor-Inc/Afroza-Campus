# Afroza Campus

Vision: Plateforme sociale éducative et professionnelle pour l'Afrique francophone et globale.

Stack technique (définie et validée):
- Frontend mobile: React Native + TypeScript
- Frontend web: Next.js + TypeScript
- Backend temps réel: Phoenix (Elixir)
- Backend APIs / logique métier: Node.js + NestJS + TypeScript
- Services haute performance: Go
- IA & Data: Python + FastAPI
- DB: PostgreSQL, Redis, SQLite (tests)
- Object storage: S3 compatible

Ce dépôt contient la structure monorepo initiale prête pour développement dans VS Code.

Consignes rapides:
- Voir `architecture.md` pour l'architecture globale
- Chaque service contient un `README.md` local expliquant rôle et démarrage
- Utiliser `docker-compose.yml` (placeholder) pour environnement local

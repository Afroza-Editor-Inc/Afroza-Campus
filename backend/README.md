# Backend — Afroza Campus

Ce dossier contient tous les services backend:

- `phoenix-realtime` — sockets, channels, présence
- `nestjs-core` — API GraphQL, auth, feed
- `go-services` — workers et services haute performance
- `python-ia` — endpoints IA & modération

Communication & orchestration:
- Redis & Pub/Sub pour events
- PostgreSQL pour stockage relationnel
- S3 compatible pour médias

Chaque service contient un README propre avec instructions de démarrage.
